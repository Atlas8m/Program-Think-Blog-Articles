<h2>★政治类</h2><br/>
<h3>◇政治类 / 政治理论 / 民主和宪政</h3><br/>
约翰·斯图尔特·密尔：《<b>代议制政府</b>》<br/>
作者是很有影响力的自由主义思想家，生活于19世纪英国。<br/>
此书是公认的，关于“议会民主制”的经典著作。<br/>
不同于某些理论著作，此书谈了很多具体的操作层面细节。<br/>
<br/>
塞缪尔·亨廷顿：《<b>第三波——20世纪后期民主化浪潮</b>》<br/>
作者是美国知名的保守派政治学家。<br/>
此书主要介绍了20世纪后期许多国家的政治转型的过程，并分析了发生转型的原因。<br/>
<br/>
<h3>◇政治类 / 政治理论 / 自由主义</h3><br/>
以赛亚·伯林：《<b>自由论</b>》(《自由四论》扩充版)<br/>
作者是很有影响力的自由主义思想家。<br/>
此书是《自由四论》的扩充版，包含作者的多篇文章。<br/>
其中最有影响力的一篇是《两种自由概念》，指出“消极自由”与“积极自由”的显著差异。<br/>
<br/>
诺齐克：《<b>无政府、国家与乌托邦</b>》<br/>
作者是美国知名哲学家，哈佛大学教授。<br/>
此书同样是很有影响力的自由主义理论著作，并且此书还专门反驳了罗尔斯的《正义论》。<br/>
此书应某读者的建议上传，特此鸣谢。<br/>
<br/>
<h3>◇政治类 / 政治理论 / 极权主义和威权主义</h3><br/>
卡尔·波普尔：《<b>开放社会及其敌人</b>》<br/>
波普尔是20世纪著名的哲学家，政治上拥护自由主义，提倡“开放社会”。<br/>
书中更多的是介绍“开放社会的敌人”，而不是“开放社会”本身。<br/>
书中着重批判了 柏拉图、黑格尔、马克思 三人的哲学体系（此三人可说是一脉相承）。<br/>
波普尔认为他们三人的哲学理论是极权主义思想的根源。<br/>
<br/>
吉列尔莫·奥唐奈 &amp; 菲利普·施密特：《<b>威权统治的转型——关于不确定民主的试探性结论</b>》<br/>
两位作者都是美国知名的政治理论学者。<br/>
此书是4卷本丛书《从威权主义统治转型——民主的前景》的最后一卷，也是“比较政治学”领域的经典著作。<br/>
此书对威权主义转型的方方面面有深刻的分析，并给出很多可操作性的建议。<br/>
希望推动天朝政治变革的网友，此书非常值得一看。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h3>◇政治类 / 政治理论 / 女权运动</h3><br/>
玛格丽特·米德：《<b>三个原始部落的性别与气质</b>》<br/>
作者是美国知名的人类学家，曾获总统自由勋章。<br/>
作者考察了新几内亚的三个原始部落，发现地理位置相近的三个部落，在文化方面（尤其是性别方面）有显著差异。<br/>
作者在此书中提出了“性别文化决定论”的学术观点。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h3>◇政治类 / 政治理论 / (其它)</h3><br/>
马基亚维利：《<b>君主论</b>》<br/>
作者是意大利的政治哲学家，文艺复兴时期的重要人物。此书是作者最出名的代表作。<br/>
此书具有广泛的影响，但是在 政治、宗教、道德 等多个领域引发了强烈的争议。<br/>
光看《君主论》，容易对作者产生偏见。要全面了解马基亚维利的思想，还需看他的另一本代表作《论李维之罗马史》。<br/>
<br/>
<h3>◇政治类 / 政治小说</h3><br/>
姜贵：《<b>旋风</b>》<br/>
作者早年加入国民党，49年去台湾。之后成为台湾著名小说家。<br/>
此书是台湾反共文学的代表作之一，和张爱玲的《秧歌》齐名。胡适和蒋梦麟对此书的评价很高。<br/>
书名包含双重含义，其一是小说主人公创建的旋风纵队，其二是作者讽刺共产运动必如旋风般消散，不会持久。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
王力雄：《<b>黄祸</b>》<br/>
作者是大陆知名作家、民族问题学者、环保人士。<br/>
此书是政治预言小说，曾入选亚洲周刊20世纪中文小说百强（第41位）<br/>
此书写于90年代初，书中描写的中国社会危机，至今一直存在，而且加深了。<br/>
<br/>
<h3>◇政治类 / 中国 / 中共历史 / 改革开放时期</h3><br/>
刘晓波：《<b>大国沉沦——写给中国的备忘录</b>》<br/>
刘晓波的名气挺大，俺就不介绍了。<br/>
本书写得比较杂，涉及了当今政坛的许多方面。<br/>
内容包括：一党独裁、权贵资本主义、维权、六四、民族主义、等。<br/>
PDF版本由热心网友精心排版并分享，特此鸣谢。<br/>
（此书之前分享过，近期某热心读者重新排版并增补了先前缺失的章节）<br/>
<br/>
<h3>◇政治类 / 中国 / 中共历史 / 文革时期</h3><br/>
潘鸣啸：《<b>失落的一代——中国的上山下乡运动(1968-1980)</b>》<br/>
作者是法国的历史学者，主要研究中国当代史。<br/>
此书详细介绍了文革时期的“上山下乡”。<br/>
<br/>
<h3>◇政治类 / 中国 / 中共历史 / “土改”运动</h3><br/>
张爱玲：《<b>秧歌</b>》<br/>
作者是著名女作家，成名于民国时期。<br/>
此书是反共文学的代表作之一，主要描写“土改”运动时期的农村。<br/>
（作者曾在1950年夏到苏北农村参加“土改”）<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
张爱玲：《<b>赤地之恋</b>》<br/>
作者是著名女作家，成名于民国时期。<br/>
此书类似于《秧歌》，也是描写“土改运动”时期的农村。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h3>◇政治类 / 中国 / 中共历史 / (其它)</h3><br/>
甘粹：《<b>北大魂——从林昭到八九民运</b>》<br/>
作者是林昭的男友，两人都被错划为“右派”。甘粹被流放新疆20年，两人经历生离死别。<br/>
此书主要包括两部分内容：其一是关于“反右运动”的历史和对林昭的纪念；其二是作者亲历“六四”的见闻。<br/>
<br/>
徐沛：《<b>无耻的洋人</b>》<br/>
作者是流亡德国的作家，持不同政见者。<br/>
此书主要揭露中共历史上一些所谓的“外国友人”是如何助纣为虐的。<br/>
<br/>
<h3>◇政治类 / 中国 / 政治人物</h3><br/>
李志绥：《<b>毛泽东私人医生回忆录</b>》<br/>
作者从1954年起被任命为毛泽东的私人医生，直到1976年毛去世为止。<br/>
此书曝光了毛太祖荒淫无度的生活，所以遭到党国的严厉查禁。<br/>
此书的内容争议较大，相关争议可参考维基百科<a href="https://zh.wikipedia.org/wiki/%E6%AF%9B%E6%B3%BD%E4%B8%9C%E7%A7%81%E4%BA%BA%E5%8C%BB%E7%94%9F%E5%9B%9E%E5%BF%86%E5%BD%95" rel="nofollow" target="_blank">这里</a><br/>
<br/>
张戎：《<b>毛泽东——鲜为人知的故事</b>》<br/>
作者是英国籍华裔作家，49年之后首位获英国博士学位的大陆公民。<br/>
此书相当于毛太祖的传记。<br/>
部分内容争议较大，好几个美国和台湾的研究学者认为某些说法缺乏可靠证据。<br/>
本书的争议可参考维基百科<a href="https://zh.wikipedia.org/wiki/%E6%AF%9B%E6%BE%A4%E6%9D%B1%EF%BC%9A%E9%AE%AE%E7%82%BA%E4%BA%BA%E7%9F%A5%E7%9A%84%E6%95%85%E4%BA%8B" rel="nofollow" target="_blank">这里</a>。<br/>
<br/>
<h3>◇政治类 / 中国 / 少数民族问题</h3><br/>
李江琳：《<b>1959：拉薩！──達賴喇嘛如何出走</b>》<br/>
天朝官方一直对1959年的“拉萨事件”语焉不详，很多网友对此事也了解甚少。<br/>
此书作者查阅大量资料，详细介绍此事的经过。<br/>
通过此书可以了解“拉萨事件”的发生是由一系列事件（比如1958年的“循化事件”）导致的矛盾集中爆发。<br/>
<br/>
<h3>◇政治类 / 中国 / 社会阶层和贫富分化</h3><br/>
李毅：《<b>中国社会分层的结构与演变</b>》<br/>
作者是旅美社会学家，此书在美国出版，俺上传的是中译本。<br/>
CHM版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h3>◇政治类 / 中国 / (其它)</h3><br/>
王小波：《<b>王小波全集</b>》<br/>
王小波的名气很大（不亚于刘晓波），俺就不介绍了。<br/>
此书是网友整理的“全集”，如果你有更全的版本，请不吝告知。 <br/>
（分享政治类电子书这么多个月，到前几天才突然想起来——尚未分享过王小波的作品 :( ）<br/>
<br/>
刘晓波：《<b>刘晓波文集</b>》<br/>
刘晓波的名气挺大，俺就不介绍了。<br/>
此书汇总刘晓波2008年10月之前的文章，目录在书末尾。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
王怡：《<b>不服从的江湖</b>》<br/>
作者是天朝的青年学者、独立中文笔会理事。曾是“关天茶舍”版主。<br/>
此书是杂文集，聊的领域很杂，其中不乏对政治素质的普及。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
胡乔英：《<b>2014大崩溃</b>》<br/>
据说此书是今年香港卖的最好的政治类禁书。<br/>
应多位读者的要求，上传了此书。<br/>
书中爆料的“中共高层绝密报告”，其真实性如何，俺持保留意见。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
这个PDF版本是逐页拍照得到的，阅读效果不太好。<br/>
<br/>
<h3>◇政治类 / 苏联</h3><br/>
以赛亚·伯林：《<b>苏联的心灵——共产主义时代的俄国文化</b>》<br/>
作者是很有影响力的自由主义思想家，也是出生于沙俄的犹太人。<br/>
此书是在伯林逝世之后，由别人汇编了他生前关于苏联的一些文章。<br/>
通过此书可以体会到苏共对文艺领域的控制和扼杀，以及对普通民众的洗脑。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h2>★历史类</h2><br/>
<h3>◇历史类 / 中国</h3><br/>
钱穆：《<b>中国历代政治得失</b>》<br/>
作者是知名历史学家，中华民国中央研究院院士。<br/>
此书是作者的演讲合集，时间上包括：汉、唐、宋、明、清 几个朝代。<br/>
内容上包括："政府的组织、职权、监察、赋税、兵役"等方面。<br/>
<br/>
吴思：《<b>血酬定律——中国历史中的生存游戏</b>》<br/>
作者是知名作家，《炎黄春秋》杂志的主编、副社长。<br/>
此书可以算是《潜规则》的姐妹篇，不过名气没那么大。<br/>
<br/>
王思誠：《<b>曠世風雷一夢痕──九十年人生經歷見證</b>》<br/>
作者长期在国民党中统任要职，49年撤往台湾。<br/>
此书提供了国民党官员的视角，来看待民国历史。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h3>◇历史类 / 德国</h3><br/>
格茨·阿利：《<b>希特勒的民族帝国——劫掠、种族战争和纳粹主义</b>》<br/>
此书着重从内政方面分析纳粹帝国。<br/>
包括纳粹如何赢得民众的广泛支持、如何解决经济和社会问题、等等。<br/>
PDF版本由热心网友分享，特此鸣谢。<br/>
<br/>
<h2>★军事类</h2><br/>
<h3>◇军事类 / 军事理论</h3><br/>
杜黑：《<b>制空权</b>》<br/>
作者是意大利将军、军事理论家。<br/>
此书发表于一战刚结束不久，书中强调了空军和制空权在未来战争的重要性。<br/>
在世界军事理论界，此书跟马汉的《海权论》齐名。<br/>
<br/>
<h3>◇军事类 / 现代军事史 / 二战</h3><br/>
汉斯·雅各布森：《<b>第二次世界大战的决定性战役</b>》<br/>
此书包括了二战的10个关键性战役的详细介绍（含作战图）。<br/>
<br/>
<h3>◇军事类 / 现代军事史 / 韩战</h3><br/>
沈志华(整理) ：《<b>朝鲜战争——俄国档案馆的解密文件</b>》<br/>
这个PDF有699页，翻译自俄国档案馆解密文件，内容包含大量绝密电报，主要是苏联、中国、北朝鲜高层之间的往来电文。<br/>
通过此书内容，有助于了解斯大林、毛泽东、金日成这三个独裁者是如何策划朝鲜战争的。<br/>
俺的博文《<a href="../../2013/08/korean-war.md">谈谈真理部对朝鲜战争的忽悠</a>》引用了此书部分电报的内容。 <br/>
<br/>
沈志华：《<b>毛泽东、斯大林与朝鲜战争</b>》<br/>
作者是历史学者，对中苏关系和冷战史有深入研究，并且是研究韩战的权威学者。<br/>
此书侧重介绍了朝鲜战争各方（美、中、苏）的高层决策。 <br/>
<br/>
王树增：《<b>远东朝鲜战争</b>》<br/>
作者是中国比较有名的军旅作家。<br/>
国内出版的朝鲜战争书籍中，这是相对客观的一本——作者没有避讳志愿军的某些失败。<br/>
缺点是：本书的文字，感情色彩太浓，理性分析不足。
