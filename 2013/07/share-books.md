<h2>★第二网盘的选择</h2><br/>
　　5月底的时候，俺发了一篇《<a href="../../2013/05/share-polity-books.md">分享政治类电子书(30本)——另征求"第2网盘"的建议</a>》。最被大伙儿看好的2款网盘分别是 <a href="https://www.dropbox.com" rel="nofollow" target="_blank">Dropbox</a> 和 <a href="https://www.mega.co.nz/" rel="nofollow" target="_blank">MEGA</a><br/>
<br/>
　　<b>容量方面</b><br/>
　　MEGA 优势明显（默认 50GB），而 Dropbox 只有可怜的 2GB（通过邀请扩容，最多到 18GB）<br/>
<br/>
　　<b>速度方面</b><br/>
　　Dropbox 优势明显。俺为了确保安全性（主要是隐匿性），必须走双重代理操作网盘。在这种情况下，Dropbox 的页面载入速度还行，而 MEGA 的页面几乎难以加载成功 :(<br/>
<br/>
　　<b>商业方面</b><br/>
　　据说 Dropbox 是全球用户数第一的网盘，不用担心它短期内倒闭。<br/>
<br/>
　　基于上述几项考虑，最后还是选了 Dropbox<br/>
<br/>
<br/>
<h2>★Dropbox 的容量问题</h2><br/>
　　Dropbox 默认的空间是 2GB，免费扩容的上限是18GB，所以再过1-2年，没准 18GB 空间也用完了，到时候俺会考虑拿 MEGA 当第三网盘。（虽然可以付费扩容，但是"在线支付"容易暴露身份，所以俺不考虑）<br/>
<br/>
　　<b>欢迎帮俺扩容</b><br/>
　　如果你今后想注册一个 Dropbox 帐号，可以点击<a href="http://db.tt/fRYKnoQ2" rel="nofollow" target="_blank">俺的邀请链接</a>。通过这个邀请链接注册的每个新账户，可以帮俺网盘增加 500兆 容量。俺的网盘容量越大，就可以分享更多的电子书。<br/>
<br/>
<br/>
<h2>★Dropbox 的同步问题</h2><br/>

<h3>◇政治类 / 政治理论 / 社会主义</h3><br/>
哈耶克：《<b>通往奴役之路</b>》<br/>
作者是英国知名经济学家和政治哲学家，奥地利经济学派最有影响力的成员。<br/>
他一贯坚持自由资本主义和个人主义、反对社会主义和集体主义。<br/>
此书阐述了：所有实行中央计划经济的社会，不管采用什么意识形态，最终都会沦为极权主义。<br/>
希特勒的纳粹主义是如此，斯大林的共产主义亦如此。<br/>
哈耶克称：要将这本书献给"所有党派的社会主义者"读一读。<br/>
