# 批判性思维扫盲：学会区分“事实”与“观点” 

-----

<div class="post-body entry-content">
<h2>★引子</h2><br/>
　　近期发了两篇博文《<a href="../../2013/04/weekly-share-48.md">每周转载：关于雅安地震的种种乱象</a>》《<a href="../../2013/04/more-supervision-less-thankfulness.md">为了地震灾民——多些“监督问责”，少些“煽情感动”</a>》，引来不少读者留言。某些网友甚至在博客留言中吵了起来。据俺观察，有些参与口水战的网友，不善于区分“事实陈述”和“观点陈述”。而区分这两者的能力，恰恰是批判性思维的基本功之一。所以，今天花点时间，扫盲一下这个话题。<a name="more"></a><br/>
<br/>
<br/>
<h2>★为啥要区分“观点陈述”和“事实陈述”？</h2><br/>
　　批判性思维的其中一个要点是：你对外来的信息不能照单全收，要经过自己的【独立】思考，要经过一定的质疑。<br/>
　　但是质疑的方式，会因对象而异。不同的信息，你必须采取不同的质疑方式。甚至有些信息（比如某些“观点陈述”），你是无需质疑的（后面会介绍为啥）。<br/>
　　但如果连“观点”和“事实”都经常搞混淆，又如何能进行正确的质疑呢？所以，区分“观点陈述”和“事实陈述”非常重要。<br/>
<br/>
<br/>
<h2>★啥是“事实陈述”？</h2><br/>
　　顾名思义，“事实陈述”就是把<b>客观</b>事实说出来。<br/>
　　比如——<q>俺这个博客是2009年1月份开张的</q>——这就是一个事实陈述。<br/>
　　那么，什么才算“事实”捏？关键看它是不是具有【客观性】。不具有客观性的东西，不能算“事实”。<br/>
<br/>
<br/>
<h2>★"事实陈述"有啥特征？</h2><br/>
<h3>◇无“对错”、无“好坏”</h3><br/>
　　前面说了，“事实陈述”就是把“事实”说出来。而事实本身就【不】存在所谓的“对错”与“好坏”。所以，如果你去评价某个“事实陈述”的对错，是没有意义的，会显得很荒诞。<br/>
<br/>
<h3>◇有“真假”</h3><br/>
　　虽然“事实陈述”没有“对错”与“好坏”的概念，但是有【真/假】的概念。这个应该好理解。符合事实的，就是“真”；反之就是“假”。<br/>
<br/>
　　这里要提醒大伙儿注意了（尤其是搞理工科的同学）。很多人喜欢用“非真即假”、“非黑即白”的眼光看问题。在某些理工科领域，或许可以这么干。但是在社会学领域，最好不要这样。因为社会学领域的某些问题，会涉及到“灰色地带”，无法用“非黑即白”的方式判定。<br/>
<br/>
<br/>
<h2>★“事实”的可验证性</h2><br/>
　　这里的“可验证性”既包括“证实”，也包括“证伪”。<br/>
<br/>
<h3>◇可验证，并且显然的</h3><br/>
　　有些事实属于很容易验证的，并且不易引发分歧。那么，针对这类事实的陈述，很容易就可以判断“真假”。比如：<q>2008年奥运会在北京举行</q>。<br/>
<br/>
<h3>◇可验证，但不显然的</h3><br/>
　　比如某人声称自己拥有多少总资产。这个陈述不是很显然，但如果非要查实，多花点力气还是有可能验证的。<br/>
<br/>
<h3>◇不可验证的</h3><br/>
　　还有一些事实，在现有的科技水平上，是无法验证的。<br/>
　　比如发生在久远年代的历史事件细节（除非发明时光机，或者当时正好有外星人到访并留下记录，否则无法验证）。<br/>
<br/>
<br/>
<h2>★哪些“事实陈述”值得争论？</h2><br/>
　　刚才提到的三个类型：第一类，通常不需要争论；第三类，即使争论也不会有结果。<br/>
　　所以，今后你如果要跟别人争执某个“事实陈述”的真假，先看看是不是属于第二类的。如果不是第二类的，还是省省吧。即使是第二类的事实陈述，也要看是否值得争论。只有性价比足够高的论题，才值得你花时间和精力去争论。<br/>
<br/>
<br/>
<h2>★关于“事实陈述”的可信度</h2><br/>
　　要判断某个“事实陈述”是否可信，办法之一是你亲自去验证。<br/>
　　但是这么干，比较费时费力。而且有的事实陈述，你是不可能亲自去验证的（比如对历史事件的陈述）。<br/>
　　当你无法亲自验证的时候，就得换种方法——分析事实陈述的依据，由此来评估其可信度高不高。<br/>
<br/>
<h3>◇依据的分类</h3><br/>
　　事实陈述的依据，大致可以分为两类。<br/>
<br/>
　　<b>第一手资料</b><br/>
　　所谓的“第一手资料”，就是说这个“事实陈述”是通过陈述者的感官直接获得信息。<br/>
　　比如亲眼所见，亲耳所闻。<br/>
<br/>
　　<b>第N手资料</b><br/>
　　所谓的“第N手资料”，就是说这个“事实陈述”是从其它途径得到信息，然后再转述的。<br/>
　　如果被转述的对象是第一手资料，那么这个陈述就是“第二手资料”；以此类推。<br/>
<br/>
　　下面分别介绍两种依据的可信度问题。<br/>
<br/>
<h3>◇第一手资料的可信度</h3><br/>
　　有句老话叫做“耳听为虚，眼见为实”。很多人受到这句老话的误导，以为第一手资料就是可信的——其实未必。<br/>
<br/>
　　<b>认知的偏差问题</b><br/>
　　所谓的“认知偏差”，就是指“亲眼所见/亲耳所闻”导致的偏差。<br/>
　　比如当你观看魔术表演的时候，虽然是亲眼所见，但看到的依然是假象。因为魔术师可以通过种种方法来诱使你产生认知偏差。<br/>
　　关于“认知偏差”的种类和成因是一个很大的话题，限于篇幅，就不展开了。有兴趣的话，可以引申阅读《<a href="../../2009/05/halo-effect.md">光环效应引发的认知误区</a>》，里面提到了“选择性失察”和“首因效应”导致的认知偏差。<br/>
<br/>
　　<b>陈述者的诚信问题</b><br/>
　　还有一种情况导致第一手资料的可信度下降，就是陈述者本身的信用问题。<br/>
　　假如某个人是个老骗子，那么此人陈述的第一手资料，可信度就要打个折扣。<br/>
<br/>
<h3>◇第N手资料的可信度</h3><br/>
　　评价第 N 手资料的可信度，需要同时考虑两方面：<br/>
　　首先，第 N 手资料需要依赖于第一手资料的可信度。如果被依赖的第一手资料有问题，那衍生出来的第 N 手资料也好不到哪里去。<br/>
　　其次，信息转手的次数越多，信息的衰减通常会越严重。<br/>
<br/>
<br/>
<h2>★啥是观点陈述？</h2><br/>
　　简单地说，就是陈述自己主观的一些东西。<br/>
　　“观点”和“事实”最大的不同就在于，“观点”受主观影响的，而“事实”【不】受主观影响。<br/>
<br/>
<br/>
<h2>★“观点陈述”的分类？</h2><br/>
<h3>◇无需思考，得出的观点</h3><br/>
　　这类观点陈述，比较常见的有：感官体验、价值观偏好。<br/>
　　感官体验，直接来自于人的动物本能。得出这类观点是不用经过大脑的，类似于条件反射。<br/>
　　价值观偏好，主要来自于每个人的生活习惯、家庭环境、文化背景、等等。一旦形成了某种价值观偏好，就如同思维定式一般。作出评价也是不经过大脑的。<br/>
<br/>
　　举例：<br/>
　　<q>今天很热</q>（感官体验）<br/>
　　<q>喜欢喝XX饮料</q>（价值观偏好）<br/>
<br/>
<h3>◇经过某种思考，得出的观点</h3><br/>
　　思考的方式有很多种：<br/>
　　最不靠谱的，就是完全靠猜测（拍脑袋），然后得到某个主观臆断。<br/>
　　好一些的方法还有：<br/>
　　依靠归纳法，总结出某个结论。<br/>
　　依靠演绎法，通过逻辑推理得到某个推论。<br/>
　　......<br/>
<br/>
<br/>
<h2>★“观点陈述”有啥特征？</h2><br/>
<h3>◇对于“感官体验”的观点</h3><br/>
　　完全不存在“真假”、“对错”、“好坏”。<br/>
　　比如同一时间同一地点，有人会觉得温度高，有人觉得温度低。你<b>不能说</b>某个人是对的，另一人是错的。<br/>
<br/>
<h3>◇对于“价值观偏好”的观点</h3><br/>
　　如果“价值观偏好”纯属【私人领域】，就跟“感官体验”类似，也没有“真假”、“对错”、“好坏”之类的说法。<br/>
　　比如喜欢什么颜色的服饰、喜欢什么口味的食物、等等。<br/>
<br/>
　　但是，对于涉及到【公共领域】（比如政治领域、道德领域）的“价值观偏好”，就会有“好坏”之分。<br/>
　　如果某个价值观在不同的历史时期、不同的文化中，都得到赞美，那么就可以称之为是“好”的；反之，都遭到批评就称为“坏”的。<br/>
　　不过捏，能称得上“好的”价值观不多，能称得上“坏的”价值观也不多。大部分公共领域的价值观，都处于中间的灰色地带。这些价值观，可能在某些历史时期或某些文化中被赞同，而在另外的历史时期或另外的文化中，被排斥。比如“性取向的问题”就属于这种中间的灰色地带。<br/>
<br/>
<h3>◇对于“经过思考得出的观点”</h3><br/>
　　这类观点，也会有“对错”之分。因为思考的过程是有可能出现偏差和谬误的。<br/>
　　比如，用演绎法进行推理的时候，可能会出现推理的逻辑谬误。<br/>
　　比如，用归纳法进行总结的时候，样本的片面性会导致结论的片面性。<br/>
　　......（还有很多种谬误，恕不一一列举）<br/>
　　关于思维方面的偏差和误区，有兴趣的话可以看看俺前几年的博文：《<a href="../../2010/07/silent-proof.md">思维的误区：忽视沉默的大多数</a>》《<a href="../../2009/02/from-surface-to-essence.md">学会透过现象看本质，即使现象有时候挺诡异</a>》<br/>
<br/>
<br/>
<h2>★哪些“观点陈述”值得质疑和争论？</h2><br/>
　　通过上述的分析，大伙儿应该比较清楚了：对于“感官体验”和“私人领域的价值观偏好”，质疑是没有意义的、争论是没有结果的。<br/>
　　可惜的是，很多天朝的网民非常热衷于这类争论。比如：不同明星的追星族会互相争论、不同球队的球迷也常互相争论、甚至不同品牌手机的粉丝也会互相争论。<br/>
<br/>
　　对于“经过思考得出的观点”，这方面是可以进行质疑和争论的。但要注意两点：<br/>
　　1. 要考虑性价比——不要花太多时间和精力去质疑和争论一些鸡毛蒜皮的细节<br/>
　　2. 反驳别人的观点要讲究境界<br/>
<br/>
　　关于第二点提到的“境界”，大伙儿可以去读一下保罗.格雷汉姆的文章《<a href="http://www.paulgraham.com/disagree.html" rel="nofollow" target="_blank">How to Disagree</a>》。考虑到某些同学不喜欢读洋文，附上如下的中文图解。<br/>
