<h2>★政治类 / 北朝鲜</h2><br/>
◇芭芭拉·德米克：《<b>我们最幸福——北韩人民的真实生活</b>》<br/>
书名具有反讽的意味。<br/>
作者是美国洛杉矶时报的记者。<br/>
作者采访了许多北朝鲜的“脱北者”，最终写成此书。<br/>
通过此书可以了解北朝鲜底层民众的日常生活。<br/>
<br/>
◇黄长烨：《<b>黄长烨回忆录</b>》<br/>
作者是北朝鲜高官，曾经担任最高人民会议议长，是主体思想方面的理论家。<br/>
1997年进入北京的南韩大使馆寻求政治避难。<br/>
通过此书可以了解北朝鲜的高层领导人以及北朝鲜的政治生活。 <br/>
<br/>
<h2>★政治类 / 苏联</h2><br/>
◇索尔仁尼琴：《<b>古拉格群岛</b>》<br/>
作者是苏联时期最著名的持不同政见者，诺贝尔奖获得者。此书是索尔仁尼琴的代表作。<br/>
作者本人长期被关押在古拉格，且出狱后又采访了几百名相关人员，最终写成此书。<br/>
书中大量披露了古拉格种种惨无人道的罪行。<br/>
如今俄罗斯已经把本书的精简版列入中学历史教材。 <br/>
<br/>
◇陈启能 等：《<b>苏联大清洗内幕</b>》<br/>
此书由中国社会科学文献出版社在1988年出版，主要揭露斯大林的几次大清洗。<br/>
本书揭露的冤案，涉及到苏联的党、政、军、科研、文艺等各个方面。 <br/>
<br/>
<h2>★政治类 / 中国 / 政治人物</h2><br/>
◇余杰：《<b>河蟹大帝胡锦涛</b>》<br/>
余杰是知名的人文及政治题材作家，独立中文笔会副会长。<br/>
此书堪称是《中国影帝温家宝》的姐妹篇，专门揭露胡锦涛的嘴脸。<br/>
某热心网友帮俺收集制作了该书，目前只找到前几个章节。<br/>
如果哪位同学有完整版，还请不吝告知哦。<br/>
<br/>
◇《<b>胡耀邦與中國政治改革——12位老共產黨人的反思</b>》<br/>
本书收录了多篇纪念胡耀邦的文章，这些文章的作者包括多位中共元老。<br/>
通过这些文章，有助于了解胡耀邦在政治改革方面的贡献。 <br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / 改革开放时期</h2><br/>
◇何清涟：《<b>现代化的陷阱</b>》<br/>
何清涟是天朝知名且敢言的经济学家，因受迫害移居美国。<br/>
此书是她的成名作，其电子版在网上流传很广。<br/>
书中介绍了经济改革引发的种种问题（包括贫富分化、国有资产流失、圈地运动、等）。<br/>
此书上次分享过 HTML 版本，此次增加了 PDF 版本。<br/>
<br/>
◇何博传：《<b>山坳上的中国——问题、困境、痛苦的选择</b>》<br/>
作者是中山大学科学哲学系副教授，发表多篇关于中国危机问题的论文。<br/>
此书1989年出版，曾一度被列为禁书。20多年后再看此书，作者的许多担忧已变成现实。<br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / 六四事件</h2><br/>
◇封从德：《<b>六四日记——广场上的共和国</b>》<br/>
封从德是六四运动中的学生领袖，通缉名单之一。<br/>
此书详细记录了他参与六四运动的整个过程。<br/>
俺在写《回顾六四系列》的过程中，也参考了它的某些内容。 <br/>
<br/>
◇北京市委办公厅：《<b>1989年北京制止动乱平息反革命暴乱纪事</b>》<br/>
此书是朝廷官方出版滴，看的时候要<b>小心被洗脑</b>。<br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / 文革时期</h2><br/>
◇徐晓 &amp; 丁东 &amp; 徐友渔：《<b>遇罗克之遗作与回忆</b>》<br/>
遇罗克是文革时期的一个小人物。<br/>
他写了《出身论》批判当时盛行的“血统论”，在国内造成很大影响。<br/>
1968年被捕，1970年被当作反革命分子处决。<br/>
此书收集了遇罗克本人的多篇文章以及相关的一些评论文章（包括攻击遇罗克的）。<br/>
<br/>
◇刘宾雁：《<b>第二种忠诚</b>》<br/>
作者曾任中国作协副主席，八六学潮后被邓小平开除党籍。六四事件后流亡美国。<br/>
在此书中，作者写道：雷锋式的人物，党让干啥就干啥，那是第一种忠诚；而敢于向党提意见，那是第二种忠诚。<br/>
此书以报告文学的形式，介绍了那些敢于向党提意见的小人物以及他们的悲惨命运。<br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / “大跃进”和大饥荒时期</h2><br/>
◇杨显惠：《<b>定西孤儿院纪事</b>》<br/>
大饥荒期间，甘肃省的定西专区是重灾区。<br/>
此书根据翔实的史料和当事人的回忆，再现了当时的惨状。<br/>
此 PDF 版本由热心网友扫描并分享。<br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / 反右运动</h2><br/>
◇章诒和：《<b>往事并不如烟</b>》<br/>
作者章诒和是“中国头号大右派”章伯钧的次女。<br/>
此书描述了史良、储安平、张伯驹、康同璧、聂绀弩、罗隆基、张庚、章伯钧等知名“大右派”的悲剧命运。<br/>
由于此书内容批评了毛太祖，在2004年大陆出版时，多处内容被删改。之后真理部禁止该书重印。<br/>
此书上次分享过 TXT 版本。<br/>
此次的 PDF 版本由某网友精心排版并分享，该版本标注了被删节的内容，特此鸣谢。 <br/>
<br/>
◇林昭：《<b>十四万言书——致人民日报编辑部信</b>》<br/>
林昭是北大才女，曾任北大校刊《未名湖》编辑和《北大诗刊》编辑。1957年“反右”期间被划为右派。<br/>
1960年在针砭时弊的《星火》杂志上发表长诗《海鸥之歌》和《普鲁米修斯受难之日》，同年10月被逮捕。<br/>
1962年初保外就医，与黄政共同起草了《中国自由青年战斗同盟》的纲领和章程。<br/>
当年底再度被捕，1968年被枪决。<br/>
她在狱中坚持自己的信仰，写了20万字的血书与日记。这本十四万言书是其中之一。<br/>
<br/>
◇《<b>林昭，不再被遗忘</b>》<br/>
此书是关于林昭的纪念文章的合集。<br/>
感谢某热心网友分享PDF版本。<br/>
<br/>
◇《<b>中华民族的自由女神——林昭罹难40年纪念文集</b>》<br/>
此书是关于林昭的纪念文章的合集。<br/>
感谢某热心网友分享PDF版本。<br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / “土改”运动</h2><br/>
◇廖亦武：《<b>最后的地主</b>》<br/>
作者是中国有名的诗人、流亡作家。<br/>
此书介绍了“土改”时期，地主的悲惨遭遇。<br/>
当时有大量的地主/富农被滥杀，至今得不到平反。<br/>
<br/>
<h2>★政治类 / 中国 / 中共历史 / 国共内战时期</h2><br/>
◇傅国涌：《<b>1949年——中国知识分子的私人记录</b>》<br/>
作者是自由撰稿人，出版过不少历史类著作。<br/>
此书收集了1949年前后，14位知名知识分子的日记和书信。<br/>
通过这些日记和书信，有助于了解那段历史。 <br/>
<br/>
<h2>★政治类 / 中国 / 外交</h2><br/>
◇吴冷西：《<b>十年论战——1956-1966中苏关系回忆录</b>》<br/>
吴冷西曾任新华社社长、《人民日报》总编、中宣部副部长等职务。<br/>
此书详细介绍了中苏关系逐步恶化的过程。<br/>
俺写的《<a href="http://program-think.blogspot.com/2012/05/three-years-famine-0.html">谈谈三年大饥荒</a>》系列博文引用过本书的内容，证明“苏联逼债”是误传。<br/>
<br/>
<h2>★政治类 / 中国 / 腐败问题</h2><br/>
◇刘宾雁：《<b>人妖之间</b>》<br/>
作者曾任中国作协副主席，八六学潮后被邓小平开除党籍。六四事件后流亡美国。<br/>
此书是文革刚结束时发表的报告文学，揭露了地方官员的重大贪污案。<br/>
<br/>
<h2>★政治类 / 中国 / 教育问题</h2><br/>
◇《<b>审视中学语文教育</b>》<br/>
此书收集了多篇文章，指出天朝官方教育存在的种种弊端。 <br/>
<br/>
<h2>★政治类 / 中国 / (其它)</h2><br/>
◇胡平：《<b>數人頭勝過砍人頭</b>》<br/>
作者是流亡美国的政治异议人士，《北京之春》杂志主编。<br/>
此书是作者本人的文集，包含了八十几篇政治评论。<br/>
PDF版本是某热心网友精心排版并分享给俺的，特此鸣谢。 <br/>
<br/>
<h2>★政治类 / 政治理论 / 非暴力抗争</h2><br/>
◇吉恩·夏普：《<b>群众性防卫——一种超军事的武器系统 </b>》<br/>
此书是吉恩·夏普又一部知名著作。<br/>
主要介绍：利用群众性防卫的方式对付内部篡权者和外部侵略者。<br/>
此书上次分享过繁体扫描版。此次增加了简体的文字版。<br/>
该版本是某热心网友分享给俺的，特此鸣谢。<br/>
<br/>
<h2>★文艺类</h2><br/>
（下面几本归到文艺类，但是跟政治有关，一并列出）<br/>
<br/>
◇王朔：《<b>美人赠我蒙汗药</b>》<br/>
作者是当代知名的作家和编剧。名气较大，俺就不多介绍了。<br/>
书的主要内容是王朔跟匿名者老侠的对话。<br/>
通过对话点评了天朝当代文化艺术，抨击大众文化的媚俗和某些知识分子的虚伪。<br/>
据说匿名对话者老侠就是刘晓波。<br/>
<br/>
◇钟祖康：《<b>中国比小说更离奇</b>》<br/>
作者曾经是香港政治评论人，2003年移居挪威。<br/>
此书描写了天朝的种种荒诞之处。如果你在天朝待久了，或许习以为常了。<br/>
在本书的自序中，作者写道：“在人的解放、平等和尊严这些信念底下，甚么爱国主义、爱党主义、民族主义都是可弃之而毫不足惜的。”<br/>
此书由热心网友分享TXT/MOBI格式，特此鸣谢。
