“那好，把你的脚挪开！你踩到我了！”<br/>
<br/>
两个老同学偶遇，然后寒暄。<br/>
“您现在在哪工作？”<br/>
“中学老师。您呢？”<br/>
“华为HR部门”<br/>
“啊，您在华为HR部门具体干什么？”<br/>
“我们负责揪出还没离职的员工中，那些对公司不满的家伙。”<br/>
“您的意思是……有的人已经离职了？”<br/>
“哦，那些人不归我们管，负责管他们的是深圳市公安局。”<br/>
<br/>
深圳看守所，某民警审问犯人：<br/>
“被羁押了多久？”<br/>
“两年了。”<br/>
“犯了什么罪？”<br/>
“没犯罪。”<br/>
“撒谎！没犯罪的，羁押251天就放出来了！”<br/>
<br/>
一个华为研发员工准备跳槽去微软，上级知道了很不高兴，找他做思想工作。<br/>
领导：“你为什么要去微软？是因为不满意工资收入？”<br/>
员工：“不是。”<br/>
领导：“是因为不满意加班时间？”<br/>
员工：“不是。”<br/>
领导：“是因为不满意公司文化？”<br/>
员工：“不是。”<br/>
领导：“那我不明白了，你没啥什么不满意的，为啥要去微软？”<br/>
员工：“因为在那里，即使我不满意，也不会被送进看守所。”<br/>
<br/>
“HR部门全都又蠢又坏！”<br/>
“你涉嫌污蔑华为被逮捕了！”<br/>
“我又没说是哪个公司的HR部门。”<br/>
“你都说了又蠢又坏，还能是哪个？！”<br/>
<br/>
谷歌员工，微软员工，华为员工在一起谈论什么是世界上最幸福的事。<br/>
谷歌员工说——<br/>
最幸福的事情就是公司福利待遇好，加班时候无比舒适。<br/>
微软员工说——<br/>
最幸福的事情就是一周可以上四天休三天。<br/>
华为员工说——<br/>
最幸福的事情是，警察半夜来敲门，对我说：李明，你因为勒索华为公司被逮捕了！<br/>
我松了一口气，对警察说：您搞错了，李明住在隔壁。<br/>
<br/>
某华为员工的鹦鹉丢了。这是只会骂人的鹦鹉，要是落到HR的手里可就糟了。<br/>
这人赶紧在华为内部论坛上发了一份声明：<br/>
“本人遗失鹦鹉一只。另外，本人不同意它关于996和251的任何观点。”<br/>
<br/>
“假设你在华为食堂里，而一个研发员工坐到你的身边并开始唉声叹气，你该怎么做？”<br/>
“立即举报给他的HR。”<br/>
<br/>
任正非在公司里发现了蟑螂，跟余承东抱怨这事。<br/>
余承东想了一会说：<br/>
在发现蟑螂的地方立个牌子，上面写【研发部】；这样一半蟑螂会过劳猝死，另一半则会被HR举报给警方逮捕。<br/>
<br/>
华为公司命令一位擅长画画的员工创作一幅名为《任正非引领全球5G建设》的大型油画作为献礼。<br/>
该员工很不情愿地接受了这份差事。<br/>
画作完成后，华为某高层前来验收，结果大吃一惊：画面上是某人在看守所被审问的场面，窗外是华为园区。<br/>
“这是什么？这男的是谁？！”高层愤怒的问。<br/>
“离职的华为员工。”员工答道。<br/>
“他对面的人呢？！”<br/>
“深圳警察。”<br/>
“那任正非在哪里？”<br/>
“任正非在引领全球5G建设。”<br/>
