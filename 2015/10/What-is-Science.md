<h2>★预备知识（这节很重要哦）</h2><br/>

<h3>◇普遍概念、个别概念</h3><br/>
　　关于这两个词汇，完备且严密的定义，会比较冗长枯燥。所以俺通过“举例子”的方式，来通俗地说明这两个词汇。<br/>
　　比如说：“国家”属于“普遍概念”；“中国”属于“个别概念”。<br/>
　　比如说：“独裁者”属于“普遍概念”；“习近平”属于“个别概念”。<br/>
　　上面的例子好理解，再来说一个稍微容易混淆的。<br/>
　　思考一下“这一届政治局的成员”属于哪种？——是“个别概念”。<br/>
　　俺个人是这样理解这两种概念的区别：“普遍概念”是无法逐一枚举其包含的实体；而“个别概念”要么是单一实体，要么其包含的实体是可以逐一枚举的。<br/>
　　有人可能会反驳说：可以逐一列举出历史上的“独裁者”，为啥还把“独裁者”当成“普遍概念”。<br/>
　　因为“独裁者”一词，既可以用来表示过去和现在的独裁者，也可以表示未来的独裁者。而未来的独裁者尚未出现，你不可能进行逐一枚举。<br/>
<br/>
<h3>◇全称陈述、单称陈述、存在陈述</h3><br/>
　　这三个概念，还有另一种说法叫做：全称命题、单称命题、存在命题。<br/>
　　同样用“举例子”的方式，来辅助大伙儿理解。<br/>
全称陈述——所有人（任何人）的身高都小于3米<br/>
单称陈述——姚明和郭敬明的身高都小于3米<br/>
存在陈述——至少存在一个人，其身高小于3米<br/>
<br/>
　　请注意，“全称陈述”对应的主体（主语）就是前面提及的“普遍概念”；“单称陈述”对应的主体就是前面提及的“个别概念”。<br/>
　　为了巩固你对“普遍概念”的印象，再来做一个练习：<br/>
“这一届政治局的成员，身高都小于3米”——这句话是哪一种陈述？<br/>
如果你【误以为】这句话是“全称陈述”，请你回过头重新复习前一个小节（普遍概念 VS 个别概念）。<br/>
<br/>
<h3>◇客观陈述、主观陈述</h3><br/>
　　“客观/主观”这两个概念是哲学领域的常见词汇，而且对这俩的定义，可说是五花八门。<br/>
　　由于本文聊的是“科学哲学”，所以这里只给出波普尔用于科学哲学的定义（这个定义是比较简洁，而且【不】牵扯到“形而上学”）。<br/>
　　波普尔说：陈述句（命题）的客观性在于——它们能通过不同的主体间进行相互检验。（出处：《<a href="https://docs.google.com/document/d/1WRVGU5qcrhhjDMbgnpYiiSiomRmgPm--ZHMYmIAhVeo/" target="_blank">科学发现的逻辑</a>》第1章）<br/>
　　因此，无法做到上述这点的，就是主观的陈述。<br/>
<br/>
<h3>◇经验、理性</h3><br/>
　　（<b>本文所说的“经验”是“<a href="https://zh.wikipedia.org/wiki/%E7%BB%8F%E9%AA%8C%E4%B8%BB%E4%B9%89" rel="nofollow" target="_blank">经验主义</a>”相关的；本文所说的“理性”是“<a href="https://zh.wikipedia.org/wiki/%E7%90%86%E6%80%A7%E4%B8%BB%E4%B9%89" rel="nofollow" target="_blank">理性主义</a>”相关的。【不要】用日常的含义来理解</b>）<br/>
　　哲学有一个很重要的领域叫做“认识论”（有时也称为“知识论”）。“经验主义”和“理性主义”是该领域的两大流派。<br/>
　　“经验主义”所说的“经验”，通俗地说就是：用人的感官通过外界进行认识。“理性主义”所说的“理性”是跟前面的“经验”相对的——也就是说，不依赖于人的感官，单靠自我思考而进行认识。<br/>
　　举例：<br/>
　　对“数论”的研究，属于“纯理性”的范畴——不需要依赖外部客观世界，也可以思考数论的问题。<br/>
　　相对的，物理学和化学的研究，需要依赖“经验”——需要通过做实验，研究外部客观世界，从而进行认识。<br/>
<br/>
<h3>◇可验证性、可重复性、可操作性</h3><br/>
　　“可验证性”指的是某个命题<b>可以通过【经验】的方式</b>进行检验。（何为“经验”，参见前一个小节）<br/>
　　为了便于大伙儿理解，俺来举一个“不可验证”的例子——隐身精灵。<br/>
　　假设某人声称地球上存在某种“隐身精灵”。这种精灵非常牛逼，你用任何技术手段都无法探知其存在。<br/>
　　那么，这种所谓的“不可知的精灵”，就是不可验证的。（下面俺会聊到—不可验证的命题，在科学上是没有意义的）<br/>
<br/>
　　“可重复性”指的是，对某个命题的验证，必须是“对多人可重复的”。<br/>
　　说个反面教材：<br/>
　　去年（2014）科学界出了一个重大丑闻（STAP细胞），关注科技动态的同学应该都听说了。日本理化所的研究员小保方晴子号称搞定了“人工多能干细胞”（轰动了干细胞研究领域）。<b>但是她论文中的实验方法，无法被任何一个生物学同行重现。</b>后来日本理化所进行内部调查，认定她的论文造假。之后小保方的指导教授笹井芳树上吊自杀。<br/>
　　在上述案例中，小保方的学术成果被质疑，关键原因在于：其他同行无法复现其实验结果——在科研领域中，这是非常致命的。<br/>
<br/>
　　“可操作性”指的是对命题的验证要能够通过切实可行的、有限的技术步骤来完成。<br/>
　　举例：<br/>
　　如何判断某个软件是否具有智能？<br/>
　　普通人可能会说：只需要让这个软件跟人比一下智力，看它是否跟人一样聪明。<br/>
　　但是，上述这种说法，【不具备】可操作性——首先，“聪明程度”是很难量化的；其次，如何进行“比智力”，也很难化解为可操作的步骤。<br/>
　　再来看计算机领域的绝对大牛图灵，他就给出了一个【可操作的】判定过程——叫做“图灵测试”（具体介绍参见“<a href="https://zh.wikipedia.org/wiki/%E5%9B%BE%E7%81%B5%E6%B5%8B%E8%AF%95" rel="nofollow" target="_blank">维基词条</a>”）<br/>
　　“图灵测试”的优点就在于其“可操作性”。<br/>
<br/>
<h3>◇证伪、证实</h3><br/>
　　通过检验，判定某个命题为“真”，称之为“证实”；反之，判定该命题为假，称之为“证伪”。<br/>
<br/>
<h3>◇演绎法、归纳法</h3><br/>
　　关于这两个词汇，完备且严密的定义，会比较冗长枯燥。所以俺通过“举例子”的方式，来通俗地说明这两个词汇。<br/>
<br/>
　　“演绎法”在某些学术文章中也被称为“分析的”。通俗的说法是“推理”。下面是一个演绎推理的三段论：<br/>
　　大前提：人都是会死的。<br/>
　　小前提：哲学家都是人。<br/>
　　结论：哲学家都是会死的。<br/>
<br/>
　　“归纳法”在某些学术文章中也被称为“综合的”。通俗的说法是“总结”。下面是举例：<br/>
　　因为每天都看到太阳从东边升起，于是就总结出（归纳出）：太阳永远都会从东边升起。<br/>
　　古代的欧洲人只看到过白天鹅，于是就总结出（归纳出）：天鹅全都是白色的。<br/>
<br/>
　　<b>“演绎法”和“归纳法”具有本质上的差异。</b><br/>
　　“演绎法”是严密的——如果“大前提和小前提”都成立，则结论【必定】成立。<br/>
　　反之，“归纳法”是【不严密】的——因为归纳法企图通过“有限次”观测来得到一个普适的结论。<br/>
　　用逻辑学的术语：“归纳法”企图通过 N 次“单称陈述”，推导出一个“全称陈述”。这必定是不严密的（始终存在例外的可能性）<br/>
<br/>
<br/>
<h2>★啥是“科学哲学”？</h2><br/>
　　“科学哲学”顾名思义，是哲学的一个分支，用来研究“科学”。<br/>
　　它研究的内容包括：科学的定义（划界问题）、科学研究的方法论、科学知识的结构、科学的目的性......<br/>
<br/>
<br/>
<h2>★“划界问题”及其流派</h2><br/>
　　关于“科学的定义”也称为“划界问题”。意识是说：划一条边界，把“科学”与“非科学”区分开。<br/>
　　（请注意，此处说的是“非科学”，而不是“伪科学”。“非科学”是“伪科学”的超集。关于这两个概念，后面还会聊到）<br/>
<br/>
　　为了解决“划界问题”，历史上出现多个不同的流派。下面大概描述一下：<br/>
<br/>
　　<b>逻辑实证主义 / 逻辑经验主义</b><br/>
　　这玩意儿洋文叫做“logical positivism”或“logical empiricism”。<br/>
　　这个流派统治了“科学哲学”很长时间。其代表人物包括“马赫、维也纳学派”等。<br/>
　　关于该流派，简而言之就是：以【可证实性】作为划界标准。<br/>
　　（实际上该流派的理论有一大砣。比如对“分析命题”和“综合命题”的区别对待......诸如此类的。因为该流派不是本文的重点，所以俺就一笔带过。）<br/>
<br/>
　　<b>证伪主义 / 批判理性主义</b><br/>
　　这玩意儿洋文叫做“falsificationism”或“critical rationalism”。<br/>
　　（为了避免歧义，本文后续部分一律采用“证伪主义”一词）<br/>
　　该流派由大名鼎鼎的<a href="https://zh.wikipedia.org/wiki/%E5%8D%A1%E5%B0%94%C2%B7%E6%B3%A2%E6%99%AE%E5%B0%94" rel="nofollow" target="_blank">卡尔·波普尔</a>开创。关于这位大牛，俺在<a href="../../2014/09/oversimplification.md">之前的博文</a>有多次提及（当时主要是聊：他对马克思理论体系的批判）。<br/>
　　关于该流派，本文后续会详述。这里先大致说一下——其核心观念是：以【可证伪性】作为划界标准。<br/>
　　（请注意，该理论同样有一大砣，“可证伪性”只是最关键的要素，而【不是】唯一要素）<br/>
　　波普尔不但用“证伪主义”创立了一个新的划界理论，而且他也指出了“逻辑实证主义”的致命缺陷。从那之后，“逻辑实证主义”的理论体系逐渐地破产。如今提及“逻辑实证主义”，大都是拿来当反面教材进行批判。<br/>
<br/>
　　“逻辑实证主义”的根本缺陷是啥捏？这个问题如果详述的话，又是一篇长博文。因为本文已经很长了。为了照顾某些阅读障碍的读者，这里俺长话短说。<br/>
　　假设某理论包含如下命题——天鹅都是白色的（这是一个全称命题）。<br/>
　　“逻辑实证主义”强调的是“证实”，它可以对这个命题做许多许多次的验证（观察许多许多的天鹅），发现果然都是白色的。但是无论你观察的次数有多么大，你都无法 100% 担保这个命题为真。因为你不可能穷举所有的天鹅。哪怕你把当今世上所有活着的天鹅都检验一遍，也不够。因为出生在未来的天鹅，你无法检验。<br/>
　　这就是“逻辑实证主义”的根本性缺陷——永远无法证实“全称命题”。<br/>
<br/>
　　<b>波普尔之后的划界理论</b><br/>
　　波普尔之后，又有若干哲学家提出其它的划界理论。简要列举其中几个：<br/>
　　托马斯·库恩提出了基于“释疑”的划界标准。<br/>
　　夏佩尔提出了基于“合理性”的划界标准。<br/>
　　萨伽德提出了“多元划界标准”，包括了“思维方式、经验验证、理论简单性、进步性”等判定标准。<br/>
　　这些人里面，以库恩的影响力最大，其代表作是《科学革命的结构》。他与波普尔生活的年代大致重叠，两人都批评对方的划界标准有缺陷。<br/>
<br/>
　　<b>对“划界问题”的消解</b><br/>
　　在波普尔之后，还有一批哲学家改换思路来解决划界问题——他们没有提出新的划界理论，而是致力于论证“划界问题”无意义（或者论证它是一个“伪问题”）。<br/>
　　这里面的代表人物包括“威拉德·蒯因、费亚阿本德、劳丹”等人。<br/>
<br/>
<br/>
<h2>★为啥只讲波普尔的“证伪主义”？</h2><br/>
　　本文俺会以波普尔开创的“证伪主义”来定义：何为科学？<br/>
　　由于前一章节已经介绍了“划界理论”的几个不同流派（“证伪主义”只是其中一个流派）。所以，肯定有读者会问：为啥只介绍这个流派。<br/>
　　为了增加本文的说服力，在聊“证伪主义”之前，先说说“证伪主义”相对于其它流派的优势。<br/>
<br/>
<h3>◇举例1——理论物理大牛对波普尔的认可</h3><br/>
　　波普尔开创了“证伪主义”之后，不但在哲学界造成轰动，甚至在理论物理领域也造成很大反响。<br/>
　　要知道，理论物理学界的风气，通常是看不起其它学科的，<b>尤其看不起哲学家</b>。下面插几段理论物理学家的八卦。<br/>
　　那个聪明绝顶的费曼（Feynman，创立“路径积分”与“费曼图”）有次给朋友写信，其中提到：<q>最近一切都好，就是我儿子让我担心——他居然想当个他妈的哲学家。</q><br/>
　　看过美剧《生活大爆炸》的同学，还可以回顾一下男主角谢耳朵（Sheldon Cooper）的形象。据说谢耳朵的原型是长期执教加州理工的盖尔曼（Murray Gell-Mann，创立夸克理论）。这个盖尔曼，不但鄙视哲学，而且鄙视其它自然科学以及实验物理学。<br/>
　　甚至连口碑极好的玻尔（Bohr，哥本哈根学派的精神领袖）也曾经发过一句牢骚：<q>我发现了，哲学家们都是在胡扯！</q>。<br/>
<br/>
　　因为这种传统，你很难找到某个哲学家，能被同时代的多个理论物理大牛认可。而波普尔恰恰做到了这一点。<br/>
　　有一年，波普尔到美国普林斯顿大学演讲，包括“爱因斯坦、玻尔、薛定谔”在内的多位理论物理学家，亲自到场听他讲演。（请注意：俺列的这3位，都是大师级的；而且他们不是光去捧场的，是真的去听讲的）。<br/>
　　理论物理学家邦迪（Hermann Bondi，创立稳态理论）甚至说过这么一段话：<q>科学中，方法是最重要的；而科学方法中，波普尔的科学方法是最重要的。</q><br/>
<br/>
<h3>◇举例2——当前的心理学权威教材依然使用“证伪主义”</h3><br/>
　　<a href="https://github.com/programthink/books" target="_blank">俺的网盘上</a>曾经分享过一本《<a href="https://docs.google.com/document/d/16-GOAoIar2zpwGwsWq0vl_0wbsnehoWYgz2srYGgzzY/" target="_blank">对“伪心理学”说不</a>》（洋文书名是：How to Think Straight about Psychology）。<br/>
　　作者基思·斯坦诺维奇是权威的心理学家，发了200多篇论文，而且论文引用率很高。此书被全球几百所高校（包括一流高校）用作教材。<br/>
　　而这本书中，用大量的篇幅讲述“证伪主义”。<br/>
<br/>
<br/>
<h2>★如何定义“科学”？</h2><br/>
　　下面，俺用逐层递进的方式，来描述“科学理论”应该符合的条件。当你搞清楚啥是“科学理论”，自然就明白啥是“科学”。<br/>
<br/>
<h3>◇首先，必须是个【理论】</h3><br/>
　　所谓的理论，起码得是【相互关联的】命题的集合。<br/>
　　举个反例：<br/>
　　像“诗歌、小说”是不能称之为“理论”的，自然也就不能称为“科学理论”。<br/>
<br/>
<h3>◇理论中的命题必须是【客观陈述】</h3><br/>
　　科学理论中的命题/陈述，必须是“【客观】陈述”。<br/>
　　（对“客观”一词，采用波普尔的定义，详见前面的“预备知识”章节）<br/>
　　如果你仔细阅读某些宗教理论，其命题/陈述不满足这条标准。<br/>
<br/>
<h3>◇理论中的命题必须是【全称陈述】</h3><br/>
　　科学理论中的命题/陈述，必须是“【全称】陈述”。<br/>
　　因为“全称陈述”才有价值，“单称陈述”和“存在陈述”，几乎没有价值。<br/>
　　（<i>数学理论中虽然会有“存在陈述”，但这些“存在陈述”都是基于“纯理性”来验证的。等到下一个小节说到“可验证性”，俺就会提到科学理论是基于“经验”来验证的——这是科学理论与数学理论的主要差异之一</i>）。<br/>
<br/>
　　下面以牛顿第二定律举例（俺博客的读者至少都上过中学吧，应该知道这玩意儿）<br/>
任何物体都符合 F=ma（这是全称陈述）<br/>
编程随想这个人符合 F=ma（这是单称陈述）<br/>
至少存在一个物体符合 F=ma（这是存在陈述）<br/>
<br/>
　　很显然，第一句话最有价值。因为该陈述指明的是【任何物体】，不管这个物体是大是小、是天然还是人造、是固态还是液态还是气态......总之都符合 F=ma<br/>
　　因此，牛顿定律可以被广泛运用到多个不同领域（比如“机械工程、空气动力学”等）。<br/>
　　反之，后面两个命题，就没啥价值——包含的信息量（知识量）太少。<br/>
<br/>
<h3>◇基于【演绎法】建立理论体系</h3><br/>
　　“科学理论”必须是基于“演绎法”来建立整个理论体系的。<br/>
　　说得更具体一点：<br/>
　　要建立科学理论，首先提出若干前提，这些前提是不言自明的（有点类似数学公理）。然后基于这些前提，进行演绎推理。得出各种各样的推论。<br/>
　　举例：<br/>
　　爱因斯坦建立“狭义相对论”主要是基于两个原理：“狭义相对性原理”和“光速不变原理”。然后由这俩推导出理论的其它内容（比如“尺缩效应”和“钟慢效应”）<br/>
<br/>
<h3>◇科学理论中的命题必须是【可验证】的</h3><br/>
　　关于“可验证”，俺强调几点：<br/>
1. 此处所说的验证，必须是“可重复”验证。<br/>
2. 此处所说的验证，是基于“经验”的验证，而【不是】基于“理性”的验证。（关于“经验、理性”二词，不要望文生义，请仔细看“预备知识”）<br/>
<br/>
<h3>◇科学理论中的命题必须是【可证伪】的</h3><br/>
　　前面已经说了——“不可验证”的理论，不能算科学理论。下面来讨论“不可证伪”。<br/>
　　“不可证伪”包括如下几种：<br/>
<b>1. 因为“不可验证”，所以无法证伪</b><br/>
（关于这种，参见前一个小节）<br/>
<b>2. 恒真命题、重言命题、套套逻辑</b><br/>
为了通俗化，俺举几个例子，都是【永远为真】的命题：<br/>
菱形有4条边（基于“定义”的恒真句）<br/>
直角三角形的三条边，要么是直角边，要么是斜边（基于“定义及枚举”的恒真句）<br/>
任何人的身高，要么比我高，要么不比我高（基于“排中律”的恒真句）<br/>
<b>3. 存在命题/存在陈述</b><br/>
前面的“预备知识”中已经介绍了何为“存在命题”。<br/>
“存在命题”是不可能证伪的。<br/>
<br/>
<h3>◇命题的【可证伪程度】越高，则理论越好</h3><br/>
　　聊完“可证伪性”，再来聊聊“可证伪”的程度。<br/>
　　先看如下两个命题：<br/>
命题1：所有三角形的内角和必定等于180度<br/>
命题2：所有三角形的内角和必定大于90度<br/>
<br/>
　　假设有人找到一个三角形，其内角和是150度——那么就可以推翻第1个命题，但却无法推翻第2个命题。<br/>
　　因此，咱们就说：这两个命题中，第1个命题更容易被推翻。用行话就是：第1个命题可证伪的程度更高。<br/>
　　如果你稍微聪明一些，自然就明白：某理论可证伪的程度越高，说明该理论给出的预测越丰富和详细；因此，该理论包含的信息量（知识量）也就越大。用通俗的话来说就是：越容易被推翻的命题，知识量越大。<br/>
　　所以，<b>那些看起来好像很容易被推翻，但是又迟迟没有被推翻的理论，那才是真正的牛B理论。</b>（广义相对论和量子力学，就属于此类）<br/>
<br/>
<h3>◇只允许某种类型的【特设性修正】</h3><br/>
　　早在波普尔写《<a href="https://docs.google.com/document/d/1WRVGU5qcrhhjDMbgnpYiiSiomRmgPm--ZHMYmIAhVeo/" target="_blank">科学发现的逻辑</a>》的时候，他就料到有些人会采用“特设性修正”来规避理论的反例，因此也就可以让理论逃避“被证伪的命运”。<br/>
　　上述这句话比较抽象，俺第 N 次举例：<br/>
<blockquote>比如俺给出一个全称陈述：“所有的人，身高都小于3米”<br/>
假如有一天，利用基因工程让某人的身高突破3米，那么俺上述那个命题，就被证伪了（找到反例了）。<br/>
为了补救上述命题，俺可以给命题加上一个定语，变成如下：<br/>
“所有未经过基因改造的人，身高都小于3米”</blockquote>　　在上述这个例子中，俺玩的把戏，波普尔称之为“利用特设性修正逃避证伪”。<br/>
　　为了防范这种下三滥钻空子的行为，波普尔在他的“证伪主义”理论中加了一个约束条件，大意如下：<br/>
<b>对理论进行“特设性修正”应该要【提高】理论的“可证伪程度”。只有这样，才算是科学理论；否则的话，就不是科学理论。</b><br/>
　　前面说了：“可证伪程度”越高，则理论的价值越高。正派的科学家，应该不断完善理论，于是理论的“可证伪程度”会上升（也就是越来越容易被推翻）；而如果通过“特设性修正”让理论的“可证伪程度”越来越低，那就相当于耍无赖了，同时理论也变得越来越没价值。<br/>
<br/>
<h3>◇上述每一个要求，单独只构成“必要条件”</h3><br/>
　　本来写到这里，这个章节就结束了——把上述这些条件汇总起来，就是判定“科学理论”的【充分条件】。<br/>
<br/>
　　但是考虑到很多人犯了一些低级错误，俺再多费点口水。<br/>
　　本章节提到了很多判定“科学理论”的依据/标准。这里面的每一个标准，【单独】都只构成【必要条件】，而不是【充分条件】。<br/>
　　再来举例：<br/>
　　比如很多占星术的理论，也具有可检验性和可证伪性。然后那帮占星术的粉丝就跳出来说，这玩意儿也是科学。他们的谬误在于——错把“必要条件”当成“充分条件”。实际上大部分占星术理论（甚至有可能是全部的占星术理论），都缺乏“演绎体系”——也就是说，他们不是基于演绎法建立起来的。<br/>
　　<b>具备上述某个单一标准的理论，【不一定是】科学。但是不具备的，就【一定不是】科学。</b><br/>
<br/>
<br/>
<h2>★“证伪主义”的积极作用</h2><br/>
　　俺觉得大致有如下几条：<br/>
1. 破除了“科学理论是绝对真理”的观念<br/>
2. 强调了基于经验证伪的重要性（“证伪”比“证实”更重要）<br/>
3. 由于鼓励了对科学理论的证伪，促进科学理论更快更好地发展<br/>
4. 提供了一种指导思想来建立科学理论（比如前面提及的：越容易“被证伪/被推翻”的理论，越有价值）<br/>
<br/>
<br/>
<h2>★啥是“非科学”？</h2><br/>
　　从语言逻辑上讲，“非科学”等价于“不是科学”。根据“排中律”，任何一个玩意儿 ，要么是“科学”，要么是“非科学”。<br/>
　　在此，俺要提醒一句：<b>千万不要把【非科学】当成贬义词</b>。<br/>
　　下面列举一些常见“非科学”：<br/>
<br/>
<h3>◇文学、艺术</h3><br/>
　　这些领域的理论，是【主观】的，而不是【客观】的。用波普尔的定义就是——无法在不同的主体间进行相互检验。<br/>
　　有句老话叫做：文无第一、武无第二。因为文艺领域是主观的，很难建立一个公认的评价标准。<br/>
<br/>
<h3>◇宗教</h3><br/>
　　前面俺已经提到了：大多数宗教理论，其命题/陈述，【不是】客观陈述。<br/>
　　另外，大多数宗教的理论的命题，有的是缺乏“可验证性”，有的命题直接在逻辑形式上就是“无法证伪”（比如前面提及的“存在命题”是不可能证伪的）。<br/>
<br/>
<h3>◇大部分的数学分支</h3><br/>
　　大部分的数学分支，都属于“纯理性”的认识领域（再次唠叨：此处的【理性】不是通常的用法/含义，请参见“预备知识”）。也就是说，这些领域不依赖于客观世界。比如：集合论、数论、算术......<br/>
　　那么，数学的哪个部分属于“科学”捏？俺认为“几何学”属于科学。几何学是研究空间的，因此几何学是需要通过【经验】进行检验的，也可以通过【经验】进行证伪的（再次唠叨：此处的【经验】不是通常的用法/含义，请参见“预备知识”）。与之相比，数学的其它分支，都是【纯理性】的。<br/>
　　实际上，传统的“欧氏几何”已经通过【经验】证伪了。<br/>
　　咱们在小学中学教的几何就是“欧氏几何”，它描述的是平坦的空间（空间曲率为零）。在平坦空间中，过直线外的一点只能作一条平行线（这就是“欧氏几何”中大名鼎鼎的平行公理）。而咱们生活的宇宙，空间曲率大于零（也称为“椭圆空间”）。在这种空间中，过直线外的一点不可能作平行线（总是会相交）。实际上还有一种空间叫做“双曲空间”，曲率为负数，过直线外一点，可以作不止一条平行线。<br/>
　　有兴趣的同学可以去看看维基百科<a href="https://zh.wikipedia.org/wiki/%E9%9D%9E%E6%AC%A7%E5%87%A0%E9%87%8C%E5%BE%97%E5%87%A0%E4%BD%95" rel="nofollow" target="_blank">关于“非欧几何”的词条</a><br/>
<br/>
　　补充说明：<br/>
　　本文发出后，关于”几何学“是否可以被证伪，产生了大量的讨论。于是俺加上这个”补充说明“<br/>
　　很多网友不同意俺的观点，他们的理由是：数学是无法证伪的，几何学属于数学，所以几何学也无法证伪。<br/>
　　这些网友的立论有问题，他们忽略了本文”预备知识“中强调过的，”可证伪性“是带有定语的，那就是”基于经验的可证伪性“。<br/>
　　再来看几何学和数学的其它分支，有何不同。<br/>
　　几何学是研究”空间结构“的。当年康德写《纯粹理性批判》的时候，认为”空间和时间“都是【先验的】（不依赖经验）。如果按照康德的说法，那么几何学跟其它数学分支本质是类似的——都是【纯粹理性】的学科。<br/>
　　但是，到了爱因斯坦发表《狭义相对论》和《广义相对论》之后，“空间和时间”已经被证明，不是“先验的”，而是“经验的”。如此一来，几何学也就跟其它数学分支有了根本差异——几何学的命题，是可以根据经验进行检验的，因此也就有可能”基于经验进行证伪“。<br/>
<br/>
<h3>◇逻辑学</h3><br/>
　　逻辑学也不是科学。<br/>
　　原因在于逻辑学中的真理是“绝对真理”，是无法证伪的（比如：矛盾律、排中律）。<br/>
<br/>
<h3>◇技术</h3><br/>
　　还有一类常见的“非科学”是技术。<br/>
　　咱们经常说的“科技”是“科学和技术”的总称。但这两者有本质区别：<br/>
科学理论是基于【演绎法】（逻辑推理）建立起来的。<br/>
技术理论是基于【归纳法】（经验总结）建立起来的。<br/>
<br/>
　　作为俺的老本行，说个软件开发领域的例子：<br/>
　　比如软件工程，就属于“技术”而不是“科学”。软件工程中强调的那些措施（每日构建、代码评审、短周期迭代...），都是基于前人的经验总结出来的，而【不是】通过逻辑推理得到的。<br/>
<br/>
　　再来聊个容易引发口水战的例子：<br/>
　　实际上中国古代就没有诞生过真正的逻辑学【体系】。诸子百家中，墨家晚期的典籍，开始出现少量朴素的逻辑思想（但远远称不上“体系”）；可惜“独尊儒术”之后，就荒废了。另外，印度传入的佛教典籍中包含的“因明”，也达不到古希腊逻辑学那么完善的程度（而且“因明”仅限于佛教领域）。<br/>
　　所以，中国古代那些能拿出来夸耀的东西，严格来讲都仅仅是【技术】，而不是【科学】。<br/>
　　比如“建筑、历法、造纸、瓷器、医术”等成就，都是基于“经验总结”，而【不是】“演绎推理”。<br/>
<br/>
<br/>
<h2>★啥是“伪科学”（pseudoscience）？</h2><br/>
　　所谓“伪科学”，通俗地说就是：本身不是科学，但是又伪装自己是科学。<br/>
　　从范畴上，“伪科学”是“非科学”的【真子集】。<br/>
　　举例：<br/>
　　按照俺一贯给党国抹黑的套路，来说说共产党经常吹捧的“科学的共产主义”（有时也称“科学的社会主义”）。<br/>
　　马教主的这套理论，冠以“科学的”作定语，显然就是标榜该理论是科学。到底是不是捏？俺偷个懒不点评了，直接引用波普尔著作《<a href="https://docs.google.com/document/d/1EkTX-vK8eX4KeqBir4nYL_A5Qe0VWymINXOdZVyq4mw/" target="_blank">猜想与反驳——科学知识的增长</a>》中的一段：<br/>
<blockquote>尽管经过其支持者不断的辩护，马克思主义的历史观最终仍然流于江湖郎中算命的水平。<br/>
在其最早的版本中（比如马克思对于“即将来到的社会革命”的特征分析），那些预言是可以被实证的，也是可以被证伪的。<br/>
然而随着历史逐渐否定其预言，后代的马克思追随者拒绝承认错误，反而重新解释马克思的理论来搭配现实证据。他们这样做虽然救了马克思理论一命，但却使得这个理论框架本身变得无法被验证真伪。<br/>
他们重新包装了理论，却也毁掉了其原本标榜的科学性质。</blockquote>　　（注：关于马克思的“历史唯物主义”，俺在如下几篇博文也聊过）<br/>

《<a href="../../2014/09/oversimplification.md">各种一元化思维的谬误——从“星座理论”到“共产主义社会”</a>》<br/>

<h2>★啥是“反科学”（anti-science）？</h2><br/>
　　科学研究本身是以思维作为主体的活动，而且科学理论是欢迎质疑和反驳的。如前所述，对科学理论的证伪，往往会触发新理论的建立。<br/>
　　而“反科学”就是采用“科学之外的力量”来打击科学活动。有的采用政治手段，有的采用宗教手段，有的采用直接暴力手段。<br/>
　　举几个例子：<br/>
　　中世纪时期，伽利略被宗教裁判所判刑（关于这事儿，列位看官应该是耳熟能详）<br/>
　　纳粹德国时期，“相对论”被当成“犹太人的物理学”加以批判，禁止传授。<br/>
　　天朝文革时期，“相对论、量子力学、大爆炸理论”被当成“资产阶级的物理学”加以批判，禁止传授。<br/>
　　在某些伊斯兰教国家，传播进化论或其它与古兰经不符的理论，会有生命危险。<br/>
<br/>
　　（列位看官是否感觉到：纳粹时期和文革时期，”反科学“的手法很像，如果你再看看俺的另一篇博文《<a href="../../2015/01/Communism-Nazism-Caesaropapism.md">人类自由的三大死敌——谈谈“共产运动、纳粹主义、政教合一”的共性</a>》，就明白为啥两者这么像了）<br/>

<h2>★列举常见谬误</h2><br/>
<h3>◇常见谬误：把科学理论等同于“绝对真理”</h3><br/>
　　如果根据“证伪主义”的划界标准，科学恰恰是【最不可能】成为“绝对真理”的。<br/>
　　科学理论总是存在“被推翻的可能性”，所以科学理论要么“已经被推翻”，要么是“暂时尚未被推翻”。<br/>
<br/>
<h3>◇常见谬误：混淆“非科学”与“伪科学”</h3><br/>
　　前面俺已经说了：“伪科学”只是“非科学”一个很小的【真子集】。<br/>
<br/>
<h3>◇常见谬误：以为“科学”总是比“非科学”更牛逼</h3><br/>
　　其实不一定。<br/>
　　比如“科学理论”需要依靠逻辑演绎法来建立。所以“逻辑学”是科学的基石。“逻辑学”应该比“科学”更牛逼，对吧。<br/>
　　但“逻辑学”属于“非科学”。<br/>
<br/>
<h3>◇常见谬误：觉得只有“科学”是重要的，“非科学”不重要</h3><br/>
　　前一条俺说了——“逻辑学”比“科学”更重要，并“逻辑学”属于“非科学”。<br/>
　　另外，人文领域全部都是“非科学”。人文领域同样是非常重要的。尤其像人文领域中的“伦理学”——重要性不亚于科学。<br/>
<br/>
<h3>◇常见谬误：已经被证伪的科学理论就没用了</h3><br/>
　　这个是【不一定】滴！<br/>
　　有些科学理论被证伪之后，确实没啥用了，比如曾经的”燃素理论“，”以太理论“。如今已经没人用到了。现在提到这两个，也都是拿来当反面教材。<br/>
　　但是，还有一些科学理论，虽然被证伪了，依然还在使用。为啥捏？<br/>
　　比如当爱因斯坦发表《狭义相对论》和《广义相对论》之后，牛顿的经典力学体系就被彻底证伪了。但是现在的工程学科（比如机械工程）还在用牛顿力学。因为在速度远远低于光速，质量远远没有达到天文学尺度的前提下，牛顿力学与实际观测的误差，非常非常小，工程上可以忽略不计。所以工程上依然用牛顿力学。<br/>
<br/>
<br/>
<h2>★对待“科学和科学理论”的正确态度</h2><br/>
<h3>◇不要盲目崇拜科学</h3><br/>
　　这种现象貌似还挺普遍——很多人把”科学理论“当成是永恒真理，并对”科学“崇拜不已。<br/>
　　首先，对”科学理论“而言——既然每一个科学理论总有可能被证明是错的，没必要把它看得太过神圣。<br/>
　　其次，对”科学“本身而言，应该把它视作某种【手段】而不应该视作”终极目标“或”崇拜对象“。<br/>
　　（关于”崇拜“的话题，还可以参考另一篇博文：《<a href="../../2014/05/fans-and-idolatry.md">不要成为“粉丝”——谈谈“偶像崇拜”的成因和危害</a>》）<br/>
<br/>
<h3>◇不要以为科学是万能的</h3><br/>
　　实际上，没有任何东西是万能的。<br/>
　　如果有人跟你说：某某东西是万能的，这人要么是傻子要么是骗子。<br/>
<br/>
<h3>◇“证伪（推翻）”比“证实”更有价值</h3><br/>
　　看完本文前面的部分，大伙儿应该明白：证伪（推翻）某个科学理论，其贡献是很大，超过了对科学理论的证实（验证）。<br/>
　　如果某人能够基于“经验”的观测，对某个理论进行证伪（找到反例），就可以把该理论推翻。推翻之后，科学家自然会去想办法创造出更好的理论，于是科学就进步了。<br/>
